# sample-app
Recommendation games for you

This app helps you access information of any games

## Features

### Home
- List of games

### Search
- Search game by name

### Favorite
- List of games added to favorite

### Games Detail
- Information about selected games
- Add/delete favorite

### Installation

1. Download the repository
2. Open the project by using Xcode
